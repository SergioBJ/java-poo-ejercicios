# Cuenta Bancaria – Java

Ejercicio básico desarrollado para practicar conceptos de programación orientada a objetos.

## Funcionalidades
- Crear una cuenta bancaria
- Depositar dinero
- Retirar dinero con validaciones
- Mostrar información de la cuenta

## Conceptos aplicados
- Clases y objetos
- Encapsulamiento
- Métodos con validaciones
- Control del estado del objeto

## Ejecución
Ejecutar la clase `EjecutarCuentaBancaria.java`.
