# Gestión de Producto – Java

Ejercicio desarrollado para practicar el control de información y validaciones mediante programación orientada a objetos.

## Funcionalidades
- Crear un producto con precio y stock
- Vender unidades con validaciones
- Reabastecer stock
- Mostrar información del producto

## Conceptos aplicados
- Encapsulamiento
- Métodos con validaciones
- Control del estado del objeto
- Uso de condicionales

## Ejecución
Ejecutar la clase `EjecutarProducto.java`.
